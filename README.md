## 🛠️ Stack Technique

Ce portfolio a été conçu pour être **moderne, léger et accessible**, en utilisant exclusivement les technologies du Web standard :

- **Frontend :** HTML5 (sémantique), CSS3 (Grid, Flexbox, Custom Properties), JavaScript (ES6+).
- **Interactivité :** [tsParticles](https://particles.js.org/) pour le background dynamique, animations au défilement (Intersection Observer API).
- **UX/UI :** Design responsive *mobile-first*, gestion native du Dark Mode/Light Mode.
- **Accessibilité :** Respect des normes ARIA, skip-links et contrastes optimisés.
